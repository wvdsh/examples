# examples

Different ways of interacting with the Wavedash SDK.

## Included examples

- `example-babylonjs`: Babylon.js Pong demo with staged Wavedash loading, SDK init, deferred events, and a local fallback shim.
- `example-zig`: Zig + WebAssembly Pong demo with staged Wavedash loading, SDK init, deferred events, and a local fallback shim for standalone browser testing.
