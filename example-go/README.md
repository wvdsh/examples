# Go

A minimal Go Pong game on Wavedash, compiled to WebAssembly.

## Prerequisites

- [Go](https://go.dev/dl/)
- [Wavedash CLI](https://github.com/wvdsh/cli/releases)

## Quick start

Replace `game_id` in [`wavedash.toml`](./wavedash.toml) with your Wavedash game ID, then:

```
./build.sh
wavedash dev
```
